# A Discord Bot

The purpose of this discord bot is to post a coded check-in into discord channels as configured. 


Additional features may include:
 - Track previous check codes for each channel to ensure codes are not reused within 30-60 days
 - Create role tagging for within each channel's daily post 


Potential future features
 - tracking of a user's session, including potentially tracking statistics of successful check-ins based on other user votes
